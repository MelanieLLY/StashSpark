# Wireframes — StashSpark

## List of Pages

- ⭐ Dashboard (Inbox)  
- ⭐ Item Detail Page  
- ⭐ Review Today Page  

## Wireframe 1: Dashboard (Inbox)

**Purpose:** Display all saved bookmarks, allow quick search/filter, and add new items.

**Layout Summary:**
- Header: app logo, search bar, user menu  
- Sidebar: quick filters (“All”, “Inbox”, “Reviewed”)  
- Main content: list of bookmark cards showing title, domain, and quick actions (view/edit/delete)  
- Floating “+ Add Item” button  

Wireframe file: `planning/wireframes/dashboard.png`

## Wireframe 2: Item Detail Page

**Purpose:** Focused view for one saved item — read summary, write notes, and manage review settings.

**Layout Summary:**
- Top: title, URL, mark as done  
- Left: summary card + “Generate Summary” button  
- Right: note editor (text or markdown)  
- Bottom: review interval dropdown  

Wireframe file: `planning/wireframes/item_detail.png`

## Wireframe 3: Review Today Page

**Purpose:** Show items due for review and allow users to mark them as completed.

**Layout Summary:**
- Header: “Review Today” title + small progress indicator  
- Main content: cards with title, snippet, and buttons (“Reviewed”, “Later”, “Skip”)  
- Progress bar for daily completion  

Wireframe file: `planning/wireframes/review_today.png`
